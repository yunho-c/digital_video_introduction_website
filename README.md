# Digital Video Introduction Website

This is a single-page website ported from the [Digital Video Introduction](https://github.com/leandromoreira/digital_video_introduction) guide, styled with a Swiss Design aesthetic.

## How to Run

You can easily serve the website locally using `npx` or `bunx`.

### Using npx

```bash
npx serve
```

### Using bunx

```bash
bunx serve
```

Once the server starts, open the provided URL (usually `http://localhost:3000`) in your browser to view the site.
